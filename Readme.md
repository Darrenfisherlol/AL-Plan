# Alex's Landscaping Client Management Tool

A simple landscaping client management application designed to help alex's landscaping businesses organize clients, schedule work, and track projects.

## Features

* View client history and project details
* Schedule landscaping services and appointments
* Manage follow-ups and customer interactions
* Track project status from planning to completion

## Technology Stack

* C# + ASP.NET Core Web API

* TS + React
* Vite + TanStack Router 

* Google Cloud Platform (GCP) 

why... bc I like it

## Goal

The goal of this project is to provide a simple solution for Alex's companies to manage clients, scheduling, and project in a SIMPLE (like really simple) application.
